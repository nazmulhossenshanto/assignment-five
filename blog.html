Important FAQ

What is Document Object Model(DOM)?
The DOM is a standard way for browsers to represent and interact with HTML and XML documents.
How do you select an element from the DOM?
To select an element from the DOM, you can use `getElementById()`, `getElementsByClassName()`, `getElementsByTagName()`, `querySelector()`, or `querySelectorAll()`. `querySelector()` selects a single element, while `querySelectorAll()` selects multiple elements.
What is event delegation in the context of the DOM, and why is it useful?
Event delegation is a technique in the DOM where a parent element handles events for its child elements, even if those child elements are added dynamically. It works by taking advantage of event bubbling, where events propagate up the DOM tree.
